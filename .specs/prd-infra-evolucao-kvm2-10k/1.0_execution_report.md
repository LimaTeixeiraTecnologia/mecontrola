## Suposições
- O textfile do node-exporter está em `/var/lib/node_exporter/textfile_collector/` (padrão Ubuntu); configurável via `TEXTFILE_DIR`.
- O container postgres no Swarm tem nome prefixado com `${STACK}_${POSTGRES_SERVICE}` (padrão `mecontrola_postgres`).
- `pgbackrest check` é o método autoritativo de verificar archive-push; falha não-zero indica problema.
- O systemd-timer é o método primário; cron `/etc/cron.d/pgbackrest` é fallback quando `systemctl is-system-running` falha.
- A guard em `render-stack.py` aplica-se apenas quando `ENVIRONMENT=production` está no env-file; staging/dev não são bloqueados.
