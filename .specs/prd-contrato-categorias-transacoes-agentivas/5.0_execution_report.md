## Contratos Implementados

| Contrato | Implementação |
|----------|--------------|
| Gate skip para income sem subcategoria | `approveUpdateCategory` retorna zero evidence quando `subID==nil` |
| Source `manual_canonical_id` → evidence determinística | `buildGateEvidence` preenche score=1.0, confidence=manual_confirmed, quality=manual_canonical |
| Version automática para manual ou ExpectedVersion<=0 | `resolveExpectedVersion` lê `VersionReader` nesses casos |
| Erro de kind mismatch bloqueia write | Gate retorna `ErrCategoryKindDirectionMismatch`; use case propaga via `fmt.Errorf("...%w", err)` |
| Erro de deprecated bloqueia write | Gate retorna `ErrCategoryDeprecated` |
| Erro de version drift bloqueia write | Gate retorna `ErrCategoryVersionDrift` |
| Erro de root sem folha bloqueia write | Gate retorna `ErrSubcategoryRequired` |
