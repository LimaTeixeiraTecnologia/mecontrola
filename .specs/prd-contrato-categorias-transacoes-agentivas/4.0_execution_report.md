## Decisões Técnicas

- **Editorial version = 7**: A migration 000001 parte de version=1 e aplica 6 UPDATEs adicionais (`SET version=2` + 5× `version+1`), totalizando 7. Todos os testes de integração usam este valor.
- **signal_type = "canonical_name"**: O CHECK do banco aceita apenas `'canonical_name','alias','phrase','merchant','segment','manual_canonical'`; `"keyword"` é rejeitado.
- **`ReconstituteEvidence`**: Bypass sem validação para reconstituição do banco; construtores de domínio continuam com validação completa.
- **Zero-value evidence em use cases**: Use cases passam `valueobjects.CategoryWriteEvidence{}` até a task 5.0 implementar o `CategoryWriteGate`.
