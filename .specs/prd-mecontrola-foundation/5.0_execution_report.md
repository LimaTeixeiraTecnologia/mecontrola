## Suposições
- O `//go:embed` foi implementado via package `migrations` (raiz do módulo) que exporta `embed.FS`, para respeitar a restrição de Go de não permitir `../` em padrões de embed. O arquivo `migrations/embed.go` exporta `var FS embed.FS` com `//go:embed *.sql`. O database package importa `migrations.FS`.
- `migrate:down` foi implementado como subcomando `migrate-down` do cobra (em vez de flags) para manter a separação limpa de subcomandos (ADR-010).
- `Port.Num()` usado na conversão de porta do testcontainers em vez de `.Int()` (API do moby/moby v1.54.2 não expõe `.Int()`).
