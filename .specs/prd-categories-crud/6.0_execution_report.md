## Decisões Técnicas

1. **Getter methods**: Adicionados métodos getter em `Category` e `Candidate` para permitir o uso de interfaces no DTO de output, facilitando a conversão de entidades para DTOs.

2. **Validação de input**: A validação do `SearchDictionaryInput` foi implementada no próprio DTO seguindo o padrão de auto-validação.

3. **Normalização de query**: Implementada no input DTO, removendo espaços, pontuação e convertendo para lowercase.

4. **Paginação**: `ListDictionary` implementa page_size padrão (50) e máximo (200) conforme RF-14a.
