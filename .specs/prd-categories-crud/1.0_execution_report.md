## Decisões Técnicas

1. **Constraint categories_parent_same_kind removida**: PostgreSQL não suporta subqueries em CHECK constraints. A validação será feita na camada de aplicação (use cases).

2. **Função immutable_unaccent criada**: A função `unaccent()` nativa não é imutável, impedindo seu uso em colunas geradas. Criada wrapper `mecontrola.immutable_unaccent(text)` marcada como IMMUTABLE.

3. **Collation pt_BR removida**: O locale pt_BR não está disponível no container Postgres padrão. A ordenação PT-BR será tratada na camada de aplicação se necessário.

4. **Versão Go**: 1.26.4 (conforme go.mod)
